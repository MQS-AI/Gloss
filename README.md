# Gloss — Word 风格页边批注插件

为 Obsidian 笔记与 PDF 添加 Word 风格的页边批注，支持高亮连线、双边分布排版、碰撞避免、附件与录音功能。

***

## 功能特性

- **页边批注卡片** — 在编辑区两侧或右侧显示批注卡片，支持 6 种内置类型 + 自定义类型和颜色
- **高亮连线** — 贝塞尔曲线 / 直线 / 阶梯线连接批注与原文高亮（上下双圆点锚定）
- **碰撞避免** — 智能垂直避让，密集批注自动均匀分布不重叠
- **附件支持** — 批注可关联 vault 内图片、视频、音频、文件（图片/视频显示缩略图，支持双击全屏预览）
- **录音附件** — 内置录音按钮，使用 Obsidian 核心录音插件录制音频并自动附加到批注
- **回复功能** — 批注卡片支持回复讨论线程
- **PDF 批注** — 支持在 PDF 页面中添加批注，选中文字自动作为引用原文，含伪高亮圆点标记
- **浮动工具栏** — Markdown 编辑器中选中文字自动弹出添加入口
- **右键菜单** — 选中文字右键快速添加批注
- **快捷键** — `Gloss: 为选中文本添加批注` 命令
- **Ribbon 开关** — 一键开启/关闭批注显示
- **原始文本引用** — 卡片内显示被批注的原文引用（可配置开关）
- **导出 PDF 批注** — 可将 PDF 批注导出为文件保存

***

## 安装

### 手动安装

1. 在 Obsidian vault 的 `.obsidian/plugins/` 下创建 `gloss` 文件夹
2. 将以下文件复制到该文件夹：
   - `manifest.json`
   - `main.js`
   - `styles.css`
3. 在 Obsidian 中进入 **设置 → 第三方插件**，刷新并启用 **Gloss**

### 开发者构建

```bash
cd obsidian-plugin
npm install
npm run build
```

## 使用方法

### Markdown 批注

**语法格式（使用 Obsidian 原生脚注）：**

```markdown
==高亮文本==[^g-42]

...

[^g-42]: general:批注内容
```

**带类型前缀：**

```markdown
==这里有误==[^g-42]

...

[^g-42]: correction:应为"xxx"
```

**带附件：**

```markdown
==图片区域==[^g-42]

...

[^g-42]: general:参考此图 | attach:路径/图片.png,路径/视频.mp4
```

**快速添加（推荐）：**

1. 用鼠标选中任意文字
2. 点击浮出的 ✏️ 按钮或右键 → `Gloss: 添加批注`
3. 选择批注类型，输入内容，可选添加附件或录音
4. 点击确认

> 附件会自动保存到 Obsidian 设置的附件文件夹中。

### PDF 批注

1. 打开 PDF 文件
2. 鼠标选中一段文字 → 工具栏出现在选区附近
3. 点击 ✏️ 按钮 → 选中文字自动填入引用原文
4. 提交后 PDF 页面上显示紫色呼吸光点标记

### 卡片操作

- **回复** — 展开输入框发送回复线程
- **编辑** — 修改批注内容、类型、附件
- **附件** — 快速添加 vault 内文件
- **解决** — 移除批注、保留原文
- **删除** — 完全删除批注和高亮

## 设置

| 设置项      | 说明                    |
| -------- | --------------------- |
| 导出文件保存路径 | 导出 PDF 批注时的 vault 相对路径，留空则保存到 PDF 同目录 |
| 防重叠智能求解器 | 动态计算卡片垂直间距，防止重叠       |
| 引导连线样式   | 贝塞尔平滑曲线 / 直连矢量线 / 阶梯线 |
| 批注侧栏排版   | 左右双边分摊 / 仅右侧单栏        |
| 显示引用原文   | 卡片中是否显示高亮文本的原文引用      |
| 卡片类型背景色  | 根据类型显示微弱的背景色区分        |
| 自定义批注类型  | 添加/编辑/删除类型标签和颜色       |

### 内置批注类型

| 类型   | 颜色        |
| ---- | --------- |
| 一般批注 | `#c084fc` |
| 错误纠正 | `#f87171` |
| 灵感记录 | `#fbbf24` |
| 内容补充 | `#60a5fa` |
| 提问质疑 | `#fb923c` |
| 总结提炼 | `#34d399` |
